Big highlights people can feel

Numbers show scale. These show why time mattered.
Big win

PrettySimpleShop works again

Not only version bump. Needed Purpur 26.1.2 update, item-name fallback fixes, and behavior cleanup. Good example of hard part being "make feature act right again," not only "make code compile."

Core plugin

MountainDewritoes got real repair work

Startup and config fixes, player-data-storage refactors, dependency cleanup. Important because this plugin holds lot of server-specific identity.

Support tool

ItemNameLab became useful infrastructure

Better translated-name handling, tests, command cleanup, component output. Less flashy, but helps old item behavior stop breaking in weird ways.

API drift

Old Bukkit or Paper assumptions no longer compile cleanly against modern server code.

Build drift

Some repos needed workflow and release cleanup before results could be trusted.

Behavior drift

Compile success was not enough. Some plugins needed real logic or config fixes too.

Context switching

Each repo had own age, style, dependencies, and weird history. Time goes there fast.
